jbehave-example
===============

JBehave example application, using the following :
 
 - Spring 
 - JBehave Annotations
 	- @AsParameterConverter
 	- @When
 	- @Given
 - Parametrised Scenarios
 - Parameter Injection
 - Outcomes Table
 - Tabular parameters
 
 to run the tests run MyStories as a JUnit test
