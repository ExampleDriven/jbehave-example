jbehave-example
===============

JBehave example application, using the following :
 
- JBehave 3.6.8
- Spring 3.1.1
- JUnit 4.8.2
- JBehave Eclipse plugin
- JBehave Annotations
	@AsParameterConverter
	@When
	@Given
- Parametrised Scenarios
- Parameter Injection
- Outcomes Table
- Tabular parameters
 
 to run the tests run mvn install or from Eclipse run MyStories.java as a JUnit test
